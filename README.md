## 📌 Deskripsi Project
Project ini dibuat untuk memenuhi penugasan akhir Mata Kuliah **Struktur Data** dengan mengangkat konsep **Perpustakaan Keliling Kota Makassar**. Sistem ini dirancang untuk membantu pengelolaan layanan perpustakaan keliling yang mengunjungi berbagai sekolah di Kota Makassar. Aplikasi ini mampu mengelola data buku, proses peminjaman dan pengembalian buku, riwayat peminjaman siswa, hingga penentuan rute perjalanan mobil perpustakaan antar sekolah

## ⚙️ Konsep yang Diimplementasikan
| Konsep Struktur Data / Algoritma | Implementasi dalam Sistem |
|---|---|
| **Array** | Menyimpan data buku perpustakaan seperti judul, penulis, tahun terbit, dan stok buku. |
| **Linked List** | Mengelola riwayat peminjaman dan pengembalian buku siswa dari berbagai sekolah di Makassar. |
| **Binary Search Tree (BST)** | Digunakan untuk pencarian judul buku dengan proses pencarian lebih cepat. |
| **Graph (Adjacency List) + Dijkstra** | Menentukan rute perjalanan mobil perpustakaan antar sekolah di Kota Makassar, termasuk total jarak, jumlah sekolah yang dilewati, dan estimasi waktu perjalanan. |
| **Sorting (Insertion Sort & Merge Sort)** | Mengurutkan data buku berdasarkan tahun terbit dari lama ke baru atau sebaliknya. Insertion Sort digunakan untuk data kecil dan Merge Sort untuk data besar. |
| **Searching (Linear Search & Binary Search)** | Linear Search digunakan untuk mencari penulis buku dan Binary Search untuk mencari stok buku. |
| **Manajemen Stok Buku** | Mengurangi stok saat buku dipinjam dan menambah stok kembali saat buku dikembalikan. |
| **Riwayat Peminjaman** | Menampilkan nama peminjam, sekolah asal, judul buku, dan status pengembalian buku. |
